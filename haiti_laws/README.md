# Haitian business-form legislation sources

This folder contains primary sources for the two main business forms in Haiti:

- Société anonyme
- Société en nom commandite

## Sources included

- `code_de_commerce_haiti_2.pdf`: Scan of the Haitian Commercial Code (Code de commerce haïtien) hosted by Archive.org. This source includes the articles on both the société en commandite and the société anonyme.
- `societe_en_nom_commandite.txt`: Cleaned transcription of articles from the Code de commerce covering the société en commandite (verify against the PDF).
- `societe_anonyme.txt`: Cleaned transcription of articles from the Code de commerce covering the société anonyme (verify against the PDF).
